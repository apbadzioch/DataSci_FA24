What is Data Visualization (Viz)¶

  • data visualization is the graphical presentation of the data

Uses of Data Visualization:¶

  1. It makes it easier and faster for a data analyst (data scientist) to discover the data patterns.
  2. To find out about the current trends in the dataset.
  3. To make more effective team communication.
  4. To offer a better understanding of data to the non-technical people such as the management team and to enable them to make more informed decisions.

Python Libraries for Data Visualization:¶

  1. matplotlib
  2. seaborn
  • both of these libraries are built ontop of NumPy
  • these libraries are said to be platform-independant:
    • they can operate on different OSs such as Windows, Linux, and MacOS
  • Python pyplot is an API that enables matplotlib to perform more effectively and interactively
  • the best source for matplotlib is: https://matplotlib.org/

Color Codes:¶
  1. 'r' for red
  2. 'g' for green
  3. 'b' for blue
  4. 'c' for cyan
  5. 'm' for magenta
  6. 'y' for yellow
  7. 'k' for black
  8. 'w' for white

4. Scatter Plots¶

  • scatter plots are used to find the relationship between different variables
  • each dot shows one observation (row)
  • the two variables should be of the same size (the same number of rows)
In [17]:
# creating the dataset
x = np.array([1,2,3,4,5,6,7])
y = np.array([0,2,6,9,3,4,5])

# creating the plot
plt.scatter(x, y, c=y, cmap='RdYlGn', s=500, edgecolors='black')
plt.colorbar()
Out[17]:
<matplotlib.colorbar.Colorbar at 0x14cb3d3d0>

Observations:

  • create the scatter plot with x and y
  • c=y colors the points based on their y values
  • cmap: the colormap used will be set to 'Red-Yellow-Green'
  • s: dot (marker) size
  • edgecolor: the color around the dots (markers)
  • plt.colorbar: adds the color bar to the plot, helping to interpret the mapping of data values to the colors

5. Histogram¶

  • one of the most popular plots
  • used to find the data distribution (frequency)
  • the two variables should be of the same size (the same number of rows)

Types of Distribution:

  1. Normal (zero skewness)
  2. Right-Skewed
  3. Left-Skewed

6. Heatmaps¶

  • used to find out about the correlation between various variables

Conclusion:¶

  • Matplotlib allows customization of plots, including adjusting colors, markers, lines, label, and tick marks
  • it provides many types of visualizations (line graphs, bar charts, scatter plots, pie charts, histograms, and heatmaps)
  • line charts are one of the most commonly used plots in Matplotlib, ideal for trends over time
  • line charts allow plotting multiple lines on the same axes
  • bar charts are great for comparing discrete or categorical data
  • pie charts are used for visualizing the parts of a whole and showing percentage distributions
  • pie charts allow for "exploding" slices to emphasize specific selections
  • scatter plots are best for visualizing relationships or correlations betwwen 2 continuous variables
  • Matplotlib allows for extensive customization of marker size (s), colors (c), and shapes based on additional data
  • histograms are ideal for visualizing the distibution of a single continuous variable
  • heatmaps are useful for visualizing data intensity across a 2D matrix
