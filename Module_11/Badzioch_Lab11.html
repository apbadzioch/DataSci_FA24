Lab 11 - Correlation, Univariate, Bivariate, Multivariate EDA - Andrew Badzioch¶

Types of EDA:

  1. Univariate: Examining one variable (feature, column)
  2. Bivariate: Examining two variables
  3. Multivariate: Examining more than two variables

Correlation:

  • relationship between variables. The value for correlation runs between -1 to 1. There are two types:
  1. Positive correlation: when the value of the variables go up and down together
  2. Negative correlation: when one variable goes up and the other goes down

1. Introduction:¶

  • context: dataset given to class by professor
  • objective: to work on a dataset and practice
  • data description: dataset is from Melbourne, AU
    • distance: The distance from the Central Business District (CBD)
    • sellerg: The real estate agent
    • method: The status of the property sale, such as sold, sold prior, or withdrawn prior to auction
    • type: The type of house, such as a house, townhouse, or unit
    • car: The number of car spaces in the property
    • region: general region of the property
    • landsize: size of the land in meters
    • yearbuilt: year the house was built
    • suburb: suburb the property is located in
    • address: address of the property
    • longitude: longitute
    • latitude: latitude
    • date: date the property was sold

Observation:

  • right skewed: positive skewed: tail is on the right: mean > median
  • normal distribution: skewness = 0: mean = median = mode
  • left skewed: negative skewed: tail is on the left: mean < median
  • plt.fugure(figsize=(x, y)): adjust the size of the plot (x units wide, y units tall)
  • plt.axvline(): creates a vertical line that will be put where needed
  • [0]: accesses the first mode, as mode() can return multiple modes in s Series

Conclusion:¶

  • Univariate: Examining one variable (feature, column)
  • Bivariate: Examining two variables
  • Multivariate: Examining more than two variables
  • correlation can only be used with numerical data, objects must be encoded (label or one_hot)
  • .corr(): computes the correlation coefficients between the numerical columns in a DataFrame
  • correlation measurs the linear relationship between two variables and ranges from -1 to +1
  • +1: perfect positive correlation (as one variable increases, the other increases)
  • -1: perfect negative correlation (as one variable increases, the other decreases)
  • 0: no linear correlation
  • the diagonal values are all 1.0 because the variable is perfectly correlated with itself
  • plt.hist(x, bins=n): displays the distributions of a dataset by grouping data into bins (intervals) and showing how many data points fal into each bin
    • x: the input data (1D, array, list, or Pandas Series)
    • bins: determines the number of bins or specifies the bin edges
  • sns.distplot(): creates a univariate distribution plot, combining a histogram with a kernel density estimate (KDE) to give a smooth representation of the data's probability density
  • sns.heatmap(): graphical representations of data where individual values are shown as colors in a matrix
    • heatmaps are commonly used for visualizing correlation matrices, frequency tables, or any 2D data.
